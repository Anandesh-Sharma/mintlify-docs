---
title: 'Delete Event'
openapi: 'DELETE /events/{event_id}'
---

This endpoint marks an event as deleted in the system. Note that this is a soft delete - the event is not permanently removed from the database but is marked with an `is_deleted` flag.

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `event_id` | string | Yes | The unique identifier of the event to delete |

## Response

The API returns the updated event object with `is_deleted` set to `true`:
