---
title: 'Get All Events'
openapi: 'GET /events'
---

This endpoint returns all events in the system, categorized by their status. Use this endpoint to retrieve a comprehensive list of events for display or further processing.

## Authentication

This endpoint requires authentication using an API key. Add the following header to your request:

```
x-api-key: YOUR_API_KEY
```

To obtain an API key, please contact our support team.

## Response

```bash
curl -X GET https://api.securegate.ai/events \
  -H "x-api-key: YOUR_API_KEY"
``` 
